## Reporting a Vulnerability

We take the security of Cloud Sentinel seriously. If you believe you have found a security vulnerability in Cloud Sentinel, please report it to us as described below.

**Do not report security vulnerabilities through public GitHub issues.**

### How to Report

Please report security vulnerabilities by emailing [security@pixelvide.com](mailto:security@pixelvide.com) with the subject line "Vulnerability Report - Cloud Sentinel".

In your email, please include:

- A description of the vulnerability.
- Steps to reproduce the vulnerability.
- Proof of concept code or screenshots.
- The version of Cloud Sentinel affected.

### Response Timeline

We will acknowledge receipt of your report within 48 hours and will aim to provide a fix or workaround within 7 days.
